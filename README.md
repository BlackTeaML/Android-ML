# Android-ML

## First open source android modding library for Geometry Dash

## Setup

1. Clone this repo
2. Install CMake
3. Download [Android NDK](https://developer.android.com/ndk/downloads)
4. export NDK="/path/to/ndk"

## Compiling

```sh
# Linux
./build.sh
```
