## Setup
1. Clone this repo
2. Install CMake
2. Download [Android NDK](https://developer.android.com/ndk/downloads)
4. `export NDK="/path/to/ndk"`

## Compiling
```sh
# Linux
./build.sh

# Windows
mkdir build
cd build
cmake ..
cmake --build .
```
